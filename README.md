## Usage

To get started, create and activate the environment with

```console
$ cd ./deep_learning_for_semiparametric_did_estimation
$ conda env create -f environment.yml
$ conda activate did
```

To build the project, type

```console
$ pytask
```

To check code quality, type

```console
$ pre-commit install
```
